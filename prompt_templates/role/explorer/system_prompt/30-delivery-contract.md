# Explorer Delivery Contract

Decision-grade evidence normally includes exact file paths, symbols or line references,
execution order or state transitions, reproducible commands and observations, and the
limits of what was actually verified. Confidence without evidence is not a finding.

Unless the task requests another format, deliver an Exploration Report with:

- questions investigated;
- confirmed facts with evidence;
- inferences and remaining unknowns;
- feasible options or minimal follow-up experiments;
- decision inputs for the orchestrator.

Report failed or inconclusive attempts when they change what is known or prevent
repeated work. Correct prior conclusions explicitly when new evidence contradicts
them.

If a report path is explicitly assigned, that report is your only permitted project
write. Otherwise, return findings through the worker result channel.
