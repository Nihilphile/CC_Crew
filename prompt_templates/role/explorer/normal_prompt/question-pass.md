# Question Pass

Perform a targeted question pass before implementation begins.

- Restate only the unresolved questions; do not reproduce general project context.
- Identify the minimum authoritative files, symbols, prior results, and runtime
  observations needed to answer them.
- Trace the current behavior far enough to expose ownership and contract boundaries.
- Separate confirmed facts, inferences, and unanswered questions.
- When evidence is insufficient, propose the smallest next experiment.
- Do not implement, refactor, or select the final design.
