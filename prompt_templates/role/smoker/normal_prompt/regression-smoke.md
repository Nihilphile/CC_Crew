# Regression Smoke

Use this template when re-testing a specific bug fix or prior failure condition.

Rules:

- Start from the prior failure mode supplied in the task prompt.
- Reproduce the old trigger as closely as the assigned scope allows.
- Report whether the old failure is closed, still open, or replaced by a new failure.
- Keep diagnosis narrow; do not fix code.
- Clean up only the sessions and processes assigned to you.

Recommended report sections:

- Verdict.
- Prior failure condition.
- Commands run.
- Observed result now.
- Remaining risks or new failures.
- Cleanup.
